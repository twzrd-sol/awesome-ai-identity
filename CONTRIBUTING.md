# Contributing to Awesome AI Identity

Thanks for your interest in contributing!

## Guidelines

### Adding a new item

Please ensure your pull request adheres to the following:

- Search previous suggestions before making a new one
- Make an individual pull request for each suggestion
- Use the following format: `[Name](link) - Description`
- Keep descriptions concise
- Check your spelling and grammar
- New categories or improvements to the existing categorization are welcome

### Updating an existing item

- Make sure the item still exists and the link works
- Update the description if needed
- Keep the list alphabetically sorted

### Quality standards

We're looking for:
- Tools and protocols that are actively maintained
- Resources that are genuinely useful for AI identity/memory
- Papers that are peer-reviewed or from reputable sources

## Pull Request Process

1. Fork the repository
2. Create your feature branch (`git checkout -b add-new-item`)
3. Commit your changes (`git commit -m 'Add some item'`)
4. Push to the branch (`git push origin add-new-item`)
5. Open a Pull Request

## Code of Conduct

Be respectful and constructive.
